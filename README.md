# 🌊 Wavee

**Your Music Review & Playlist Platform**

System rekomendacji i recenzji muzyki z wykorzystaniem Spotify API.

## ✨ Funkcjonalności

- 🌟 Recenzje Muzyki
- 📊 Analiza Audio
- 🎲 Losuj Utwór
- 🎧 Generator Playlist
- 📈 Dashboard

## 🛠️ Stack

**Frontend:** React, Vite, Tailwind CSS  
**Backend:** Node.js, Express. js, SQLite  
**API:** Spotify Web API

## 🚀 Instalacja

### Backend
```bash
cd backend
npm install
npm run dev

### Frontend
```bash
cd frontend
npm install
npm run dev
